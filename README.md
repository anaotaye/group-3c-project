# Expense Tracker API

## 📌 Description

This is a REST API built using Node.js and Express.js to manage expenses.

---

## ⚙️ Setup Instructions

1. Install dependencies:

```
npm install
```

2. Run the server:

```
node server.js
```

---

## 📡 Endpoints

### GET /expenses

Returns all expenses

### POST /expenses

Adds a new expense

Example:

```
{
  "amount": 5000,
  "category": "Food",
  "date": "2026-04-27",
  "description": "Lunch"
}
```
